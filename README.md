## 🛠️ Lzq.Extensions 核心能力矩阵

| 扩展库 | 能力 | MES 使用场景 |
|--------|------|-------------|
| `Lzq.Core` | ServiceBase 基类 / ApiResult / 自动 DI | 所有服务基类 |
| `Lzq.Extensions.SqlSugar` | ORM / CodeFirst / 多租户 | 9 个模块的数据持久化 |
| `Lzq.Extensions.Redis` | 缓存 / 分布式锁 | 会话缓存、验证码 |
| `Lzq.Extensions.JWT` | 认证 / 授权 | 用户登录、API 鉴权 |
| `Lzq.Extensions.Serilog` | 结构化日志 | 全链路日志追踪 |
| `Lzq.Extensions.NSwag` | Swagger 文档 | API 文档自动生成 |
| `Lzq.Extensions.EventBus` | 事件总线 / CQRS | 领域事件发布 |
| `Lzq.Extensions.RabbitMQ` | 消息队列 | 异步任务处理 |
| `Lzq.Extensions.AI` | **AgentSkill 引擎** | AI 对话 + Skill 调用 |
| `Lzq.Extensions.ExternalHttpApi` | HTTP 客户端 | 外部 API 调用 |
